## python code slicing operator

Python code example: python code slicing operator

```python
#Use slicing operator

def rotate(arr, d):
    return arr[d:] + arr[:d]
  
if __name__ == '__main__':
    arr = [1, 2, 3, 4, 5]
    arr = rotate(arr, 2)
    print((arr))


```
