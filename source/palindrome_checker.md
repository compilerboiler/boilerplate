## palindrome checker

Python beginners example: palindrome checker

```python
word=str(input('Enter the word to check:'))
reverse_word=word[::-1]


if word==reverse_word:
    print('the word ',word,'is a palindrome')


else:
    print('Just an ordinary word')

    


```
