## python conversions

Python beginners example: python conversions

```python
#!/usr/bin/python

#num is of type int
num = 8 
#eight is of type str
eight = str(num)
print("eight is of type ", type(eight))

#num8 is of type int 
num8 = int(eight)
print("num8 is of type ", type(num8))

t = True
strt = str(t)
print("strt is of type ", type(strt))

```
