## SumTwoNumbers

Python beginners example: SumTwoNumbers

```python
#Python Program for adding two numbers
num1 = input("Enter first number: ")

#Treating numbers as integers
sum = int(num1) + int(num2)

print("The sum is : {0}".format(sum))
```
