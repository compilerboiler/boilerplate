## golang-scan

Golang beginners example: golang-scan

```go
package main

import "fmt"

func main() {
	var s string
	fmt.Print("please insert a string an press enter ")
	fmt.Scan(&s)
	fmt.Printf("read string \"%v\" from stdin\n", s)
}

```
